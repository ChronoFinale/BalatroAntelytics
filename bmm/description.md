# Antelytics

Records every decision in a Balatro run — blinds, hands, discards, shop buys,
packs, and PvP scoring — to a single compressed file. Open it in the
[Antelytics viewer](https://www.antelytics.gg) to step back through the whole run
ante by ante.

Built with **Balatro Multiplayer** in mind (it captures both players' PvP scores
and lives, who won each PvP blind, and the match result), but it works on solo
runs too.

You can also annotate any step in the viewer and download the run + your notes as
one file to hand to a coach — they reply on the steps and send it back.

**Run files** land in the mod's `log` folder; load them at
[www.antelytics.gg](https://www.antelytics.gg).

**Requirements:** Steamodded. **Balatro Multiplayer is optional** — it's only
needed to capture PvP (opponent scores, lives, match result); solo runs record
without it.
